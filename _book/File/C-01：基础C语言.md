### task 4

阅读下列程序，完成以下任务：

```c
#include <stdio.h>

int fibo(int);

int main() {
    int n;
    scanf("%d%d", &n);
    printf("%d", fibo());
    return 0;
}

int fibo(int n) {
    return n <= 2 ? 1 : fibo(n - 1) + fibo(n - 2);
}
```

1. `fibo`函数的作用是什么？
2. 描述`int fibo(int);`各个部分的含义。
